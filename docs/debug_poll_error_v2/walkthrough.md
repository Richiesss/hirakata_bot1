# OOMエラー修正の確認内容

## 変更内容
1.  **Gunicornの設定変更** (`gunicorn_config.py`)
    *   ワーカー数を `multiprocessing.cpu_count() * 2 + 1` (17) から **4** に固定しました。
    *   これにより、システム全体のメモリ消費量を大幅に削減しました。

2.  **AI分析機能の最適化** (`features/ai_analysis.py`)
    *   `compute_embeddings` メソッドのデフォルトバッチサイズを **32** から **8** に変更しました。
    *   これにより、一度の推論で消費されるピークメモリを削減しました。

*   プロセスリスト(`ps aux`)を確認し、意図した通りのワーカー数で動作していることを確認しました。
